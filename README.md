# Responsive-Portfolio-Website
Responsive personal portfolio website built using HTML5, CSS3 (Grid &amp; Flexbox), and JavaScript with form validation and GitHub Pages deployment.

---

## 📱 Responsive Design

- Layout implemented using CSS Grid & Flexbox
- Media queries for multiple screen sizes
- Mobile-first responsive approach

---

## 📬 Contact Form Validation

- JavaScript-based client-side validation
- Required field checks
- Email format validation
- Error message display
